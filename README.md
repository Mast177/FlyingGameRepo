# Flying Game (game development backup repository)
This repository serves as a backup and development history log for the flying arcade game which I (Mast177) am currently developing.\
This project is being made so that I can become more familiar with the game development process and working with Godot.

## Features
- Multiple levels to play and highscore system
- Multiple enemys
- Mutiple enemy spawn and movement patterns
- Player ship upgrade system (speed, health, damage, weapon count, shield protection)
- Level unlock and level select system
- 2 original soundtracks
- Sound effects
- Fullscreen option
- Music and SFX volume control

## Technologies used
- Godot Engine 4.6
- GDScript coding language
- GDShaders

## Instalation:
1. Download all files within main branch and add to a new project folder
2. Open Godot project manager
3. Select Import
4. Select previously made project folder
5. Select Import in new window
6. Select project from project manager and click edit to edit or run to simply run game

## Playable Versions:
- Play in browser: [Space Shooters on Itch.io](https://mastcookie.itch.io/space-shooters)
- Download for Windows: [Space Shooters on Windows](Playable_Versions/Space_Shooters_Windows.zip)
- Download for Linux: [Space Shooters on Linux](Playable_Versions/SpaceShooters_Linux.zip)

## Project Structure
- Raw assests (sprites, scripts, audio tracks, animation files) are included in the Assets folder
- Resources used by scenes (player upgrade data, level wave data) are included in the Resources folder
- Game objects (player ship, enemies, menus, levels, .tscn files) are included in the Scenes folder
- Godot engine plugins are included within the addons/AS2P folder
